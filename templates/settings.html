Using SSH key authentication

Sign API requests with your SSH private key. Include these headers on every request:

Signature-Input: sig1=("@method" "@path" "@authority" "date");keyid="<YOUR_FINGERPRINT>";alg="ed25519";created=<UNIX_TIMESTAMP>
Signature: sig1=:<BASE64_SIGNATURE>:
Date: <RFC7231_DATE>

For POST/PUT/PATCH requests also include:

Content-Digest: sha-256=:<BASE64_SHA256_OF_BODY>:

The signature covers the RFC 9421 signature base constructed from the listed components. See the docs or RFC 9421 for details.
